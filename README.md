## 🌟 Overview

**Awesome Test** is a robust, fast, and highly responsive web application engineered to transform how students prepare for competitive exams. Designed with a distraction-free interface, it allows users to seamlessly attempt subject-specific mock tests, dive deep into Previous Year Questions (PYQs), and evaluate their performance in real-time.

Whether you are building fundamental logic or simulating high-pressure exam environments, this platform brings the test center directly to your screen.

---

## ✨ Standout Features

- 🧠 **Dynamic Subject Modules:** Specialized environments for **Mathematics**, **Computer Science** (including Boolean Algebra), and **Analytical Reasoning**.
- 📜 **Extensive PYQ Archive:** Instant access to crucial past papers to decode exam patterns and strategize effectively.
- 🏆 **Real-Time Leaderboard:** A gamified ranking system that fuels healthy competition and tracks peer-to-peer progress.
- 🔐 **Secure User Profiles:** Complete authentication flow including Secure Login, Password Reset, and individualized user dashboards.
- ⚡ **Lightning Fast UI:** Built with raw web technologies to ensure zero-lag performance and a fluid experience across all devices.

---

## 💻 Technical Arsenal

This platform leverages core web technologies to deliver a lightweight yet powerful user experience:

- **Frontend:** HTML5, CSS3 (Advanced Styling & Layouts)
- **Logic & DOM Manipulation:** Vanilla JavaScript (ES6+)
- **Hosting:** GitHub Pages
